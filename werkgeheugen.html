    Werkgeheugen

    RAM staat voor Random Access Memory is het type geheugen dat samenwerkt met de procesor. Er wordt in de RAM word tijdeelijk gegevens en programma-instructies opgeslagen. Het wordt ook wel het werkgeheugen van de computer genoemd. RAM bestaat uit zeer snelle geheugencellen die alleen onder spanning werken. Bij het uizetten van de computer is het RAM daarom leeg. RAM is dus altijd eigenlijk een tijdelijk geheugen. Rom staat voor read only memory het is een gedeelte van een interne geheugen. Hierin staan gegevens die alleen gelezen kunnen worden. Het Rom speelt een grote rol tijdens het opstarten van je computer. Er wordt bijvoorbeeld aan de computer opgedragen om een aantal hardwaretests uit te voeren. Verder wordt de opdracht gegeven tot het laden van de eerste onderdelen van het besturingssysteem. De gegevens in Rom worden niet gewist, ook niet als je de pc dus uitzet. Er zijn 2 belangrijke verschillen met RAM en ROM In RAM veranderen de gegevens doorlopend. De gegevens in RAM worden kwijtgeraakt als je de computer uitzet, dat is dan ook alsje de stroom uitvalt of de accu bijvoorbeeld leeg is. Als je op dat moment bezig bent zonder dat er tussentijds iets opgeslagen is, dan ben je het gemaakte werk dus helemaal kwijt. We noemen het RAM dan ook een vorm van vluchtig geheugen. Het ROM is nietvluchtig.
